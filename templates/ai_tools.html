Prompting Tips
Be Specific

Instead of "write code", try "write a Python function that validates email addresses using regex"

Give Context

Explain your use case, skill level, and what you've already tried for better responses

Set a Role

"Act as a senior developer reviewing my code for security vulnerabilities"

Ask for Steps

"Walk me through this step-by-step" helps get detailed, structured explanations

Iterate & Refine

"Make it shorter" or "Add error handling" - keep refining until it's right
